### p10137

```

예를 들면, 1508원이 총 합이고 인원이 5명이면 2명은 301원을내고 3명은 302원을 낸다.(편의를 위해 값의 * 100을 했습니다.)

돈의 이동을 적게 하기 위해서 인원의 배열을 정렬해주고
301  301   302    302    302
a <   b  <  c   <  d   <  e

각인원의 값이 자기가 내야할 돈보다 많으면 그값을 뺀값을 모두 더하여 /100 을 해주면 된다.

```
